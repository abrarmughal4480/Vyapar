# Vyparr Frontend

This is the frontend (Next.js) for the Vyparr business management app.

# Devease Digital - Business Management App

A complete business management solution built with Next.js 14 and TypeScript.

## Features

- **Authentication**: Login/Signup with business setup
- **Dashboard**: Comprehensive business overview
- **Inventory Management**: Track products, stock levels, and categories
- **Sales Management**: Record sales, track orders, and manage invoices
- **Customer Management**: Maintain customer database and purchase history
- **Reports & Analytics**: Business insights and performance metrics
- **Mobile-First Design**: Responsive design with bottom navigation

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Hooks
- **Storage**: LocalStorage (demo)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
